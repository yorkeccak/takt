# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, please report it responsibly by emailing **contact@valyu.ai**. Do not open a public issue.

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

This policy applies to the Takt open-source codebase. For issues related to the Valyu API itself, please contact contact@valyu.ai directly.
